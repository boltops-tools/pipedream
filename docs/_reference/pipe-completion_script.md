## Usage

    pipe completion_script

## Description

Generates a script that can be eval to setup auto-completion.

To use, add the following to your `~/.bashrc` or `~/.profile`

    eval $(pipe completion_script)

```
[--verbose], [--no-verbose]  
[--noop], [--no-noop]        
```
