## Usage

    pipe deploy PIPELINE_NAME

## Description

Deploy pipeline.


## Options

```
b, [--branch=BRANCH]             # git branch
                                 # Default: master
    [--stack-name=STACK_NAME]    # Override the generated stack name. If you use this you must always specify it
    [--wait], [--no-wait]        # Wait for operation to complete
                                 # Default: true
    [--verbose], [--no-verbose]  
    [--noop], [--no-noop]        
```
